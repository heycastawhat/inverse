# Inverse
---
## Hide photos, quickly and simply.
---
## Features
- Zap (Hide) images on any website
- Remembers zapped images
- Unzap (restore) hidden images
---
## Usage
- Click the Inverse icon in your browser toolbar.
- Click **Zap** to hide images (cursor becomes a crosshair), then click any image to hide it.
- Click **Unzap** to restore hidden images.
